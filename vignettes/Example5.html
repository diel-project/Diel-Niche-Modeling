We will consider a set of observed data on a species of interest from camera traps. For each observation, the time of the event is recorded and then converted into radians.


# Define a year variable
  diel.data$min_year = year(as.POSIXct(diel.data$min_date, format = "%m/%d/%Y"))

# Extract winter data in 2018 for the Virginia Opossum
  deer = subset(diel.data, min_year=="2020" & scientificName=="Odocoileus virginianus")

# Visualize the data
  head(deer)
